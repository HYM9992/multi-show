# AGENTS.md

## Project
这是一个中文单页项目展示网页，用于展示“面向自闭症谱系障碍儿童的多模态情绪识别与社交训练系统”。

## Goal
产出一个可本地直接打开的高质量前端展示页，适用于比赛展示、项目介绍、老师评审与合作展示。

## Stack
- index.html
- style.css
- main.js
- 无后端
- 尽量不用复杂框架

## Design Rules
- 风格：科技、温和、专业、可信
- 不允许大面积纯黑背景
- 文字必须清晰可读
- 动效适度，不夸张
- 中文优先
- 页面像正式产品，不像练习模板

## Engineering Rules
- 优先保证结构清晰、语义化、可维护
- CSS 需要分区注释
- JS 只做必要交互
- 避免引入不必要依赖
- 注意桌面端与移动端适配
- 注意性能，避免重动画和无意义特效

## Workflow
- 复杂修改前先给出简短实施思路
- 先搭结构，再补视觉，再补交互，再做优化
- 每轮修改后说明改动文件和下一步建议
- 如发现现有代码冲突，优先最小改动修复

## Done Means
- 页面可直接运行
- 主要模块完整
- 导航与锚点可用
- 无明显错位、遮挡、溢出
- 文字清晰
- 响应式基本正常
